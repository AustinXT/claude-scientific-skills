# 科学技能

## 科学数据库

- **AlphaFold DB** - 来自 DeepMind 的综合 AI 预测蛋白质结构数据库，提供 2 亿多个高置信度蛋白质结构预测，涵盖 UniProt 参考蛋白质组等。包括置信度指标（用于每个残基置信度的 pLDDT、用于成对准确性估计的 PAE）、结构质量评估、预测对齐误差矩阵和多种结构格式（PDB、mmCIF、AlphaFold DB 格式）。支持通过 REST API 进行编程访问、通过 Google Cloud Storage 批量下载以及与结构分析工具集成。无需实验结构测定即可实现基于结构的药物发现、蛋白质功能预测、结构基因组学、比较建模和结构生物信息学研究
- **ChEMBL** - 由 EMBL-EBI 维护的具有药物样特性的生物活性分子的综合手动管理数据库。包含 2M+ 独特化合物、19M+ 生物活性测量、13K+ 蛋白质靶标以及来自 90K+ 出版物的 110 万+ 检测。提供详细的化合物信息，包括化学结构（SMILES、InChI）、生物活性数据（IC50、EC50、Ki、Kd 值）、靶标信息（蛋白质家族、通路）、ADMET 特性、药物适应症、临床试验数据和专利信息。具有 REST API 访问、Web 界面、可下载数据文件以及与其他数据库（UniProt、PubChem、DrugBank）的集成。使用案例：药物发现、靶点识别、先导化合物优化、生物活性预测、化学生物学研究和药物再利用
- **ClinPGx** - 临床药物基因组学数据库（PharmGKB 的后续版本），提供基因-药物相互作用、CPIC 临床指南、等位基因功能、药物标签以及精准医学和个性化药物治疗的药物基因组学注释（整合了 PharmGKB、CPIC 和 PharmCAT 资源）
- **ClinVar** - NCBI 的基因组变异及其临床意义的公共档案，包括标准化分类（致病性、良性、VUS）、电子实用程序 API 访问以及用于变异解释和精准医学研究的批量 FTP 下载
- **ClinicalTrials.gov** - 全球范围内进行的临床研究的综合登记（由美国国家医学图书馆维护），可通过 API v2 访问，按条件、干预、地点、申办者、研究状态和阶段搜索试验；检索详细的试验信息，包括资格标准、结果、联系人和地点；导出为 CSV/JSON 格式进行分析（公共 API，无需身份验证，~50 请求/分钟速率限制）
- **COSMIC** - 癌症体细胞突变目录，世界上最大的体细胞癌症突变数据库（数千种癌症类型的数百万个突变、癌症基因普查、突变特征、结构变异和耐药性数据）
- **DrugBank** - 全面的生物信息学和化学信息学数据库，包含详细的药物和药物靶标信息（9,591 多个药物条目，包括 2,037 个 FDA 批准的小分子、241 种生物技术药物、96 种营养保健品、6,000 多个实验化合物），每个条目有 200 多个数据字段，涵盖化学结构（SMILES、InChI）、药理学（作用机制、药效学、ADME）、药物间相互作用、蛋白质靶标（酶、转运蛋白、载体）、生物途径、外部标识符（PubChem、ChEMBL、UniProt）以及药物发现、药理学研究、相互作用分析、靶标识别、化学相似性搜索和 ADMET 预测的理化特性
- **ENA（欧洲核苷酸档案）** - 核苷酸序列数据和元数据的综合公共存储库，具有用于访问序列、组件、样本、研究和读取的 REST API；支持高级搜索、分类查找和通过 FTP/Aspera 批量下载（速率限制：50 请求/秒）
- **Ensembl** - 基因组浏览器和生物信息学数据库，提供 250 多个脊椎动物物种的基因组注释、序列、变异和比较基因组学数据（2025 年发布 115），并具有用于基因查找、序列检索、变异效应预测 (VEP)、直向同源物查找、装配图谱 (GRCh37/GRCh38) 和区域分析的综合 REST API
- **FDA 数据库** - 通过 openFDA API 全面访问所有 FDA（食品和药物管理局）监管数据库，涵盖药物（不良事件、标签、NDC、召回、批准、短缺）、医疗器械（不良事件、510k 许可、PMA、UDI、分类）、食品（召回、不良事件、过敏原跟踪）、动物/兽药（物种特异性不良事件）和物质（UNII/CAS 查找、化学结构、分子数据）用于药物安全研究、药物警戒、法规遵从性和科学分析
- **GEO（基因表达综合）** - NCBI 的高通量基因表达和功能基因组学数据的综合公共存储库。包含来自微阵列、RNA-seq、ChIP-seq、ATAC-seq 和其他高通量实验的 264K+ 研究、8M+ 样本和 PB 级数据。提供标准化数据提交格式（MINIML、SOFT）、通过 Entrez 编程实用程序（电子实用程序）和 GEOquery R 包进行编程访问、批量 FTP 下载以及基于 Web 的搜索和检索。支持数据挖掘、荟萃分析、差异表达分析和交叉研究比较。包括精选数据集、带有实验设计的系列记录、平台注释和示例元数据。使用案例：基因表达分析、生物标志物发现、疾病机制研究、药物反应研究和功能基因组学研究
- **GWAS 目录** - NHGRI-EBI 已发表的全基因组关联研究目录，其中包含精选的 SNP-性状关联（数千项研究，全基因组显着关联 p≤5×10⁻⁸）、完整汇总统计数据、用于变异/性状/基因查询的 REST API 访问以及用于遗传流行病学和精准医学研究的 FTP 下载
- **HMDB（人类代谢组数据库）** - 全面的代谢组学资源，包含 220K+ 代谢物条目、详细的化学/生物数据、浓度范围、疾病关联、途径和用于代谢物鉴定和生物标志物发现的光谱数据
- **KEGG** - 京都基因和基因组百科全书，整合基因组、化学和系统功能信息的综合数据库资源。提供通路数据库（KEGG PATHWAY 包含 500 多个参考通路、代谢通路、信号通路、疾病通路）、基因组数据库（KEGG GENES 包含来自 5,000 多个生物体的基因目录）、化学数据库（KEGG COMPOUND、KEGG DRUG、KEGG GLYCAN）和疾病/药物数据库（KEGG DISEASE、KEGG DRUG）。具有途径富集分析、基因到途径映射、化合物搜索、分子相互作用网络、直向同源物识别（KO - KEGG Orthology）、跨数据库的 ID 转换和可视化工具。支持 REST API 访问、用于路径映射的 KEGG Mapper 以及与生物信息学工具的集成。使用案例：通路富集分析、代谢通路重建、药物靶标识别、比较基因组学、系统生物学和基因功能注释
- **代谢组学工作台** - NIH 共同基金代谢组学数据存储库，包含 4,200 多个经过处理的研究、标准化命名法 (RefMet)、质谱搜索和用于访问代谢物结构、研究元数据、实验结果和基因/蛋白质代谢物关联的综合 REST API
- **OpenAlex** - 包含 2.4 亿多学术作品、作者、机构、主题、来源、出版商和资助者的综合开放目录。提供完整的文献计量数据库，用于学术文献检索、引文分析、研究趋势跟踪、作者出版物发现、机构研究成果分析和开放获取论文识别。具有无需身份验证的 REST API（每天 10 万个请求，电子邮件每秒 10 个请求）、高级过滤（出版年份、引文、开放获取状态、主题、作者、机构）、聚合/分组功能、研究随机抽样、批次 ID 查找（DOI、ORCID、ROR、ISSN）以及全面的元数据（标题、摘要、引文、作者身份、主题、资助）。支持所有科学领域的文献综述、文献计量分析、研究成果评估、引文网络分析和学术数据库查询
- **开放目标** - 综合治疗靶标识别和验证平台，集成遗传学、组学和化学数据（2 亿+证据字符串、靶标疾病与评分的关联、可处理性评估、安全责任、来自 ChEMBL 的已知药物、GraphQL API），用于药物靶点发现、优先级排序、证据评估、药物再利用、竞争情报和机制研究
- **NCBI 基因** - 来自 NCBI 的综合基因特定数据库，提供有关 500 多种生物体基因的精选信息。包含基因命名法（官方符号、别名、全名）、基因组位置（染色体位置、外显子、内含子）、序列（基因组、mRNA、蛋白质）、基因功能和表型、通路和相互作用、直向同源物和旁系同源物、变异数据（SNP、突变）、表达数据以及对 200 多个外部数据库（UniProt、Ensembl、HGNC、OMIM、反应组）。支持通过 E-utilities API（Entrez 编程实用程序）和 NCBI 数据集 API、批量下载和 Web 界面进行编程访问。实现基因注释、比较基因组学、变异解释、通路分析以及与其他 NCBI 资源（PubMed、dbSNP、ClinVar）的集成。使用案例：基因信息检索、变异注释、功能基因组学、疾病基因发现和生物信息学工作流程
- **蛋白质数据库 (PDB)** - 蛋白质、核酸和生物大分子 3D 结构数据的全球存储库。包含 200K+ 通过 X 射线晶体学、NMR 光谱和冷冻电子显微镜实验确定的结构。提供全面的结构信息，包括原子坐标、实验数据、结构质量指标、配体结合位点、蛋白质-蛋白质界面和元数据（作者、方法、引文）。具有先进的搜索功能（按序列、结构相似性、配体、有机体、分辨率）、REST API 和 FTP 访问、结构可视化工具以及与分析软件的集成。支持结构比较、同源性建模、药物设计、结构生物学研究和教育用途。由 wwPDB 联盟（RCSB PDB、PDBe、PDBj、BMRB）维护。使用案例：结构生物学研究、药物发现、蛋白质工程、分子建模和结构生物信息学
- **PubChem** - 由 NCBI 维护的世界上最大的免费化学信息数据库。包含 110M+ 独特的化合物、270M+ 生物活性测试结果、300M+ 化学结构和 1M+ 专利。提供全面的化合物信息，包括化学结构（2D/3D 结构、SMILES、InChI）、理化性质（分子量、logP、氢键供体/受体）、生物活性数据（分析、靶标、途径）、安全性和毒性数据、文献参考和供应商信息。具有 REST API（PUG REST、PUG SOAP、PUG View）、具有高级搜索、批量下载以及与其他 NCBI 资源集成的 Web 界面。支持化学相似性搜索、子结构搜索、基于属性的过滤和化学信息学分析。使用案例：药物发现、化学生物学、先导化合物识别、ADMET 预测、化学数据库挖掘和分子特性分析
- **PubMed** - NCBI 的综合生物医学文献数据库，包含 MEDLINE、生命科学期刊和在线书籍的 3500 万多条引用。提供对摘要、全文文章（如果有）、MeSH（医学主题标题）术语、作者信息、出版日期和引用网络的访问。具有高级搜索功能，包括布尔运算符、字段标签（作者、标题、期刊、MeSH 术语、出版日期）、过滤器（文章类型、种类、语言、出版日期范围）以及通过电子邮件警报保存的搜索。支持通过电子实用程序 API（Entrez 编程实用程序）进行编程访问、批量下载、多种格式（RIS、BibTeX、MEDLINE）的引文导出以及与参考管理软件的集成。包括用于开放获取全文文章的 PubMed Central (PMC)。使用案例：文献检索、系统评论、引文分析、研究发现以及及时了解科学出版物
- **Reactome** - 用于生物过程和分子相互作用的精心策划的通路数据库（2,825+ 人类通路、16K+ 反应、11K+ 蛋白质），使用内容服务和分析服务 API 进行通路富集分析、表达数据分析和物种比较
- **STRING** - 蛋白质-蛋白质相互作用网络数据库（5000+基因组、59.3M蛋白质、20B+相互作用），具有功能富集分析、相互作用伙伴发现以及来自实验数据的网络可视化、计算预测和文本挖掘
- **UniProt** - 蛋白质序列、注释和功能信息的通用蛋白质资源（UniProtKB/Swiss-Prot 审阅条目、TrEMBL 未审阅条目），具有 REST API 访问权限，用于跨 200 多个数据库的搜索、检索、ID 映射和批量操作
- **USPTO** - 美国专利商标局数据访问，包括专利检索、商标查找、专利审查历史 (PEDS)、审查意见通知书、转让、引文和诉讼记录；支持PatentSearch API（基于ElasticSearch的专利搜索）、TSDR（商标状态和文档检索）、专利/商标分配API以及用于全面知识产权分析的其他专用API
- **ZINC** - 由 UCSF 维护的用于虚拟筛选和药物发现的商用化合物的免费数据库。包含来自 100 多家供应商的 2.3 亿多种可购买化合物，采用可立即对接的 3D 格式（SDF、MOL2）以及预先计算的构象异构体。提供化合物信息，包括化学结构、供应商信息和定价、理化性质（分子量、logP、氢键供体/受体、可旋转键）、药物相似过滤器（Lipinski 的五规则、Veber 规则）和子结构搜索功能。具有多个化合物子集（类药物、类先导化合物、类片段、天然产物）、用于特定筛选活动的可下载子集，以及与分子对接软件（AutoDock、DOCK、Glide）的集成。支持基于结构和基于配体的虚拟筛选工作流程。使用案例：虚拟筛选活动、先导化合物识别、化合物库设计、高通量对接和药物发现研究

### 基因组学和生物医学数据云平台
- **DNAnexus 集成** - 用于与 DNAnexus 云平台配合进行基因组学和生物医学数据分析的综合工具包。涵盖构建和部署应用程序/小程序 (Python/Bash)、管理数据对象（文件、记录、数据库）、运行分析和工作流程、使用 dxpy Python SDK 以及配置应用程序元数据和依赖项（dxapp.json 设置、系统包、Docker、资产）。支持处理 FASTQ/BAM/VCF 文件、生物信息学管道、作业执行、工作流程编排和平台操作（包括项目管理和权限）

### 实验室自动化

### 电子实验室笔记本 (ELN)
- **LabArchives 集成** - 用于与 LabArchives 电子实验室笔记本 (ELN) REST API 交互的工具包。提供对笔记本（备份、检索、管理）、条目（创建、评论、附件）、用户身份验证、站点报告和分析以及第三方集成（Protocols.io、GraphPad Prism、SnapGene、Geneious、Jupyter、REDCap）的编程访问。包括用于配置设置、笔记本操作和条目管理的 Python 脚本。支持多区域 API 端点（美国、英国、澳大利亚）和 OAuth 身份验证

### 工作流程平台和云执行
- **LatchBio 集成** - 与 Latch 平台集成，用于构建、部署和执行生物信息学工作流程。为使用 Python 装饰器创建无服务器生物信息学管道、部署 Nextflow/Snakemake 管道、管理云数据（LatchFile、LatchDir）和结构化注册表（项目、表、记录）、配置计算资源（CPU、GPU、内存、存储）以及使用预构建的 Latch Verified 工作流程（RNA-seq、AlphaFold、DESeq2、单细胞分析、CRISPR 编辑）提供全面支持。通过全面的数据管理，在可扩展的云基础设施上实现自动容器化、UI 生成、工作流版本控制和执行

### 显微镜和生物图像数据
- **OMERO 集成** - 使用 Python 与 OMERO 显微镜数据管理系统交互的工具包。提供对 OMERO 服务器中存储的显微镜图像的全面访问，包括数据集和筛选数据检索、像素数据分析、注释和元数据管理、感兴趣区域 (ROI) 创建和分析、批处理、OMERO.scripts 开发以及用于结构化数据存储的 OMERO.tables。对于处理高内容筛选数据、多维显微镜数据集或协作图像存储库的研究人员至关重要

### 协议管理与共享
- **Protocols.io 集成** - 与protocols.io API 集成，用于管理科学协议。允许以编程方式访问协议发现（按关键字、DOI、类别搜索）、协议生命周期管理（创建、更新、使用 DOI 发布）、分步过程文档、通过工作区和讨论进行协作开发、文件管理（上传数据、图像、文档）、实验跟踪和文档以及数据导出。支持 OAuth 身份验证、协议 PDF 生成、材料管理、线程注释、工作区权限和机构协议存储库。对于方案标准化、可重复性、实验室知识管理和科学协作至关重要

## 科学包

### 生物信息学和基因组学
- **AnnData** - 用于处理带注释的数据矩阵的 Python 包，专门为单细胞基因组数据设计。提供高维数据的高效存储和操作以及相关注释（观察/细胞和变量/基因）。主要功能包括：基于 HDF5 的 h5ad 文件格式，用于高效 I/O 和压缩，与 pandas DataFrames 集成以实现元数据，支持稀疏矩阵 (scipy.sparse) 以提高内存效率，分层数据组织（X 表示主数据矩阵，obs 表示观察注释，var 表示变量注释，obsm/varm 表示多维注释，obsp/varp 表示成对矩阵），以及与 Scanpy、scvi-tools 和其他工具的无缝集成单细胞分析包。支持延迟加载、分块操作以及与其他格式（CSV、HDF5、Zarr）之间的转换。使用案例：单细胞 RNA-seq 数据管理、多模式单细胞数据（RNA+ATAC、CITE-seq）、空间转录组学以及任何需要高效存储和操作的高维注释数据
